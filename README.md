## Getting Started

### Clone the repository

```bash
git clone https://github.com/Ruchitha98/Sampath-Weerasekara-Portfolio.git
```

### Install dependencies

```bash
yarn install
```

### Run development server

```bash
yarn dev
```

### Build for production

```bash
yarn build
```
